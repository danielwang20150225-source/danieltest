# 工作日报技能

## 触发条件
- 定时任务触发（每日9点）
- 用户手动要求生成日报

## 功能
1. 读取前一天的对话记录（webchat、飞书等渠道）
2. 按主题分类整理
3. 生成结构化工作日报
4. 通过飞书推送给用户

## 输出格式

```markdown
# 2026-03-26 工作日报

## 📊 对话概览
- 对话渠道：webchat、飞书
- 对话数量：N条
- 主要话题：XXX、YYY

## 🔍 详细内容

### 1. [话题名称]
- 来源：飞书/webchat
- 用户：daniel
- 内容：...
- 处理结果：...

### 2. [话题名称]
- ...

## 📈 待跟进
- [ ] TODO 1
- [ ] TODO 2

## 💡 学到的新知识
- 知识点1
- 知识点2

## 🛠️ 执行的工具/操作
- 工具1：执行了XXX
- 工具2：执行了YYY

---
*由虾仁自动生成*
```

## 实现方式

1. 通过 `sessions_history` 获取主会话的对话历史
2. 通过飞书消息记录获取飞书渠道的对话
3. 解析并分类整理
4. 通过 `message` 工具推送到飞书

## 注意事项
- 需要跨渠道收集对话（webchat + 飞书）
- 对话内容可能需要从不同session获取
- 飞书推送需要使用用户的 open_id
