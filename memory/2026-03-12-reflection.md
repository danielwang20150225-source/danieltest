## 三、需要沉淀为Skill的技巧

### 1. 飞书文档创建SOP
```
每次创建飞书文档时：
1. 用create创建（只传title）
2. 立即用append追加内容
3. 避免content参数（会创建空文档）
```

### 2. ClawHub Skill安装流程
```
1. 先用inspect查看安全评分
2. 确认无风险后再install
3. 如果被标记suspicious，用--force
4. 遇到Rate limit就等待，不要连续尝试
```

### 3. MCP服务排查流程
```
1. 先查官方文档是否有配置说明
2. 尝试配置文件中的mcpServers字段
3. 如果不支持就放弃，不要强行配置
```
