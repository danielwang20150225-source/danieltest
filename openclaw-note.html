🦞 OpenClaw 小龙虾

开源个人AI助手 · 从入门到精通

1 什么是 OpenClaw？

一款开源个人AI助手，可以运行在本地设备上，通过集成 WhatsApp、Telegram、iMessage、Discord 等聊天软件与用户对话。

它不只是"回答问题"，而是能 自己完成任务 —— 读取信息、处理内容、自动执行。

2 核心架构

🔷 Gateway 网关

负责与聊天平台通信，是整个系统的枢纽

🤖 Agent 智能体

通过 AI 模型处理任务和对话，可接入 Claude、GPT、Deepseek

🛠️ Skills 技能模块

扩展功能，如浏览器自动化、邮件处理、文件操作等

🧠 Memory 记忆系统

持续记住用户对话内容、偏好设置，长期+短期记忆

3 为什么这么火？

超强理解能力 超强创造力 多任务处理 人性化对话 自我改进

4 支持渠道

WhatsApp Telegram iMessage Discord Slack 飞书

5 应用场景

👤 个人AI助手

随时通过聊天工具召唤，7×24小时在线

💼 办公自动化

整理邮件、管理日程、处理工作内容

💻 产品开发

完整前后端开发能力，如魔法简历项目

📚 学习辅助

问答、知识查询、编程辅助
